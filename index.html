My name is Fernando de Orbegoso and my dream has always been to create. Create great tools to help your business grow, or perhaps a simple application to make your day easier. My passion is development and my goal is to help small business access the benefits of modern technology by breaking down barriers and high cost. My mission is to provide my expertise in both business and development to help small business grow and achieve their goals. These are some of my applications.

PRORATER is an iOS application developed using SWIFT to assist property managers calculate prorated rental amounts in any given day of the month.

Weather Dashboard is an Javascript application to display current weather and 5 day forecast.

Work Day Scheduler is an Javascript application that creates a daily calendar that the user can utilize to create a daily task list.

Food Truck Locator is an Javascript application that uses a server side API that allows for the search and mapping of food truck locations in a city.
